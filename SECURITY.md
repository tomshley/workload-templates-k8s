# Security Policy

## Supported Templates

This repository provides Kubernetes workload and component templates intended for reuse in application deployments.
Only the most recent versions of templates are actively maintained.

## Reporting a Vulnerability

If you discover a security vulnerability:

- **Do not** open a public issue
- Report it privately via a GitLab confidential issue or direct contact with Tomshley LLC

Please include:
- Template name and path
- Description of the issue
- Steps to reproduce if applicable

## Response Expectations

Tomshley LLC will:
- Acknowledge receipt of the report
- Assess the impact
- Publish fixes or mitigations as appropriate

Timelines may vary depending on severity and scope.
